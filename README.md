# 📊 AI Business Intelligence Dashboard

An AI-powered analytics system that automatically analyzes business data, generates insights, and forecasts future sales.

## 🚀 Features

• Upload business dataset
• Automated business insights
• Interactive analytics dashboard
• AI sales forecasting
• Visual data exploration

## 🧠 Tech Stack

Python
Pandas
Plotly
Streamlit
Prophet

## 📂 Project Structure

ai-business-intelligence
│
├── dashboard

├── src
│   ├── insight_engine.py
│   └── forecast_engine.py
├── data
├── notebooks
├── models
├── requirements.txt
└── README.md

## ▶️ Run Locally

Install dependencies:

pip install -r requirements.txt

Run dashboard:

python -m streamlit run dashboard/app.py
